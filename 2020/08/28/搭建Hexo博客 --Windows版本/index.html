搭建Hexo博客 --Windows版本

Posted on 2020-08-28 In Diary

本文简介

  • 本文使用Hexo+Github在Windows上搭建博客，总结了自己常用的设置，以供大家参考。
  • 文章参考：Hexo文档、NexT使用文档、Theme NexT。

环境准备

  1. 安装Node.js和git：
    Node.js 官网：https://nodejs.org/zh-cn/
    git 官网：https://git-scm.com/

    检查Node、npm和git版本号（git需在git bash上查看）

  2. 创建一个Github账号，Github官网：https://github.com/

  3. 创建代码库：
    点击右上角的小加号->New repository->”Repository name”中输入代码库名称（名称必须为<yourname>.github.io）->勾选”Public”->Create repository

  4. 开启gh-pages功能（默认是打开的）：
    打开新创建的库，打开设置，找到GitHub Pages设置，显示Your site is published at https://<yourname>.github.io/表示已打开。

个性化配置

  1. 更改站点配置文件：
    站点配置文件中设置了博客的标题、作者、文章布局和主题使用等配置，根据提示可以简单修改，这里不再赘述。想知道更多请参考Hexo文档。

  2. 主题可以浏览Hexo主题，下载主题到Theme目录下并修改站点配置文件的Theme值为该主题即可启用。

NexT

NexT也是基于Hexo的，由于其界面简单，功能全面等优点被广泛使用，接下来讲一讲它的个性化配置。

  • 主题：修改主题配置文件_config.yml的Scheme值更改主题，修改darkmode的布尔值可切换主题模式。

  • 图标：修改favicon可以更改博客的图标。

  • 菜单栏：修改menu自定义博客菜单栏。

  • 社交栏：修改social可在博客中添加自己的联系方式，如邮箱、Twitter等。
