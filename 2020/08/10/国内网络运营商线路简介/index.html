国内网络运营商线路简介

Posted on 2020-08-10 Edited on 2020-08-28 In Note

电信

  • 回国或出国，必定经过北京/上海/广州。

  • 电信网络分为三个品级：

    • 163骨干网（CN1）；

    • CN2 GT（Global Transfer 半程 CN2）；

    • CN2 GIA（Global Internet Access 纯 CN2/全程 CN2）。

163 骨干网

  • 电信 163 网连接国际网络，会在高峰时段，在路由出海前的最后一跳，根据优先级，策略性地人为丢包，以减轻对主网的负担(QOS)；

  • 中国电信的 163 网络，一般会经过市级 → 省级 → 国际出口 → 境外接入点 AS号为 4134 的路由节点，这些路由节点的 IP 地址开头一律是 “202.97”。

CN2 GT

  • CN2 GT 从市级 → 省级 → 国际出口这一段走的是 163 网络，国际出口 → 境外接入点这一段汇入 CN2 网络，返程同理，偶尔可能会走联通的国际网络。

CN2 GIA

  • CN2 GIA 从市级 → 省级 → 国际出口 → 境外接入点的过程中，全程走 AS 号为 4809 的路由节点，这些路由节点的 IP 地址开头一律是 “59.43”；

  • 部分偏远地区的电信用户访问境外 CN2 GIA 线路的服务器时，会经过 IP 地址为 “202.97” 的 AS4314 路由一跳，并入到已布设 CN2 网络的重点市级节点，随后再全程走 CN2 网络。

联通

  • 回国或出国，必定经过北京/上海/广州；

  • 回国或出国，走 AS4837（联通 169 网络）或 AS9929（A网）路由。

169 网络

  • 联通从市级 → 省级 → 国际出口 → 境外接入点的过程中，全程走 AS 号为 4837 的路由节点，这些路由节点的 IP 地址开头一律是 “219.158”。

A 网

  • 从市级 → 省级 → 国际出口 → 境外接入点的过程中，全程走 AS 号为 9929 的路由节点，这些路由节点的 IP 地址开头一律是 “218.105 /210”，部分偏远地区，会经过 169 网络汇入到 A 网；

  • A 网仅会出现在国内，去程出海后一律并入到 169 网络上；回程必须要经过 169 网络，走到国内部分才转入到 A 网。

移动

  • 中国移动进出国际网络，在国内经过的绝大部分流量，均由 AS9808 网络承载；

  • 在北京/上海/广州均有进出海光缆；

  • 流经上海移动的流量，会转交给国内其他运营商进行国际通信；

  • 移动的国内部分走 AS9808 路由（221.176 /221.183），在部分地区访问部分 VPS，会并入到铁通 AS9314/电信 163 网络/联通 169 网络的路由节点。
