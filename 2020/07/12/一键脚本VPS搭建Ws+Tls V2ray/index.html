连接 VPS ，安装 V2ray

  1. 如果有必要，重装 VPS 系统。

  2. 取得IP地址Public IP address和端口号SSH Port并记录密码。

  3. Windows连接 VPS 需要下载 Xshell 或 PuTTY 。

  4. 以 Xshell 为例，File|New：

    Name随便写；

    Protocol默认SSH；

    Host填入IP地址；

    Port Number填入端口号。

    点击确认，在弹出窗口中输入用户名和密码，用户名默认为root。若没有出现弹窗，说明 VPS 被墙。

配置 V2ray

  1. 传输协议选择 4. WebSocket + TLS ，需要域名；
  2. 自动配置 TLS ，开启 网络伪装 和 路径分流 ，伪装网站和分流路径默认就好；
  3. 端口使用五位数以上的以免冲突，一般默认就好；
  4. 不开启广告拦截；
  5. 无需配置Shadowsocks；
  6. 配置完成后保存配置信息，使用 V2ray 可以再次查看或更改配置。
